Guideline 2.4: Navigable
Success Criterion 2.4.7 Focus Visible

The purpose of this success criterion is to help a person know which element has the keyboard focus. This can be achieved in different ways.

One option is to use CSS to create a two tone focus indicator. In CSS, two-color indicators can be implemented by combining the outline and box-shadow properties with the :focus or :focus-visible pseudo-classes.

Here is an example of using CSS to create a focus indicator with a white inner border, and a black outer border.

*:focus {
/* inner white indicator */
outline: 2px #FFF solid;
outline-offset: 0;
/* outer black indicator */
box-shadow: 0 0 0 4px #000;
}
